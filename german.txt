
The 26 Standard Letters

| Letter | Name in German | Pronunciation (approximate)                                                  |
| ------ | -------------- | ---------------------------------------------------------------------------- |
| A      | ah             | like “ah” in *father*                                                        |
| B      | beh            | like English “bay”                                                           |
| C      | tseh           | “tsay” (before e, i, ä, ö, ü = “ts” sound; before a, o, u = “k”)             |
| D      | deh            | like English “day”                                                           |
| E      | eh             | like “ay” in *say*, but shorter/tenser                                       |
| F      | eff            | like English “f”                                                             |
| G      | geh            | hard “g” like *go* (never soft like *giant*)                                 |
| H      | hah            | like English “h”                                                             |
| I      | ih             | like “ee” in *see*, but shorter/tenser                                       |
| J      | jot            | like English “y” in *yes*                                                    |
| K      | kah            | like English “k”                                                             |
| L      | ell            | like English “l”                                                             |
| M      | emm            | like English “m”                                                             |
| N      | enn            | like English “n”                                                             |
| O      | oh             | like “o” in *go* (but without the final glide)                               |
| P      | peh            | like English “pay”                                                           |
| Q      | kuh            | “koo” (always followed by “u”, pronounced like “kv”)                         |
| R      | err            | guttural, uvular trill or fricative in throat (different from English “r”)   |
| S      | ess            | like English “s” (before vowels, often “z” sound, e.g. *Sonne* = zon-ne)     |
| T      | teh            | like English “t”                                                             |
| U      | uh             | like “oo” in *food*                                                          |
| V      | fau            | pronounced like “f” (e.g. *Vater* = father)                                  |
| W      | weh            | pronounced like English “v”                                                  |
| X      | iks            | like English “x”                                                             |
| Y      | ypsilon        | usually like German “ü” or English “i” (in loanwords often like “y” in *my*) |
| Z      | tsett          | pronounced “ts” (like in *cats*)                                             |

Counting
=================================================================================================
Here’s a clear guide to counting in German with spelling, pronunciation, and notes on patterns:

=================================================================================================
0–20
=================================================================================================
Number	German	Pronunciation (approx.)
0	null	nool
1	eins	ains
2	zwei	tsvai
3	drei	drai
4	vier	feer
5	fünf	foonf (with rounded “ü”)
6	sechs	zeks
7	sieben	ZEE-ben
8	acht	ahkt (ch = rough throat sound)
9	neun	noyn
10	zehn	tsayn
11	elf	elf
12	zwölf	tsvolf (rounded “ö”)
13	dreizehn	DRAI-tsayn
14	vierzehn	FEER-tsayn
15	fünfzehn	FOONF-tsayn
16	sechzehn	ZEHK-tsayn (drop the “s”)
17	siebzehn	ZEEP-tsayn (shortened from sieben)
18	achtzehn	AHKT-tsayn
19	neunzehn	NOYN-tsayn
20	zwanzig	TSVAN-tsig (“ig” = “ich” sound)
==================================================================================================

21–29
==================================================================================================
German flips the order:

21 = einundzwanzig (“one and twenty”)

22 = zweiundzwanzig (“two and twenty”)

…

29 = neunundzwanzig

So the structure is [unit] + und + [tens].
==================================================================================================

Tens
==================================================================================================
Number	German	Pronunciation
30	dreißig	DRAI-sikh (ß = “ss”)
40	vierzig	FEER-tsikh
50	fünfzig	FOONF-tsikh
60	sechzig	ZEHK-tsikh
70	siebzig	ZEEP-tsikh
80	achtzig	AHKT-tsikh
90	neunzig	NOYN-tsikh
100	(ein)hundert	(Ain)-HOON-dert
==================================================================================================


Hundreds & Thousands
==================================================================================================
101 = hunderteins (hundred-one)

235 = zweihundertfünfunddreißig (two hundred five-and-thirty)

1,000 = eintausend

2025 = zweitausendfünfundzwanzig
==================================================================================================


✅ Pattern to remember:

Units come before tens (e.g. 47 = siebenundvierzig → “seven and forty”).

Tens/hundreds/thousands behave consistently once you learn the roots.
